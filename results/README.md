# Computational Results

This directory contains raw computational results generated by the experiment scripts.

Typical contents include:

* file names of instances
* objective values
* running times

The results reported in the paper can be reproduced by executing

```bash
python ../experiments/run_maxsize.py xLOLIB_150
python ../experiments/run_constructive.py xLOLIB_150 40
python ../experiments/run_constructive.py xLOLIB_250 40
python ../experiments/run_local_search.py xLOLIB_150 40
python ../experiments/run_local_search.py xLOLIB_750 40
python ../experiments/run_local_search.py xLOLIB_1000 40
```

The generated CSV files can be processed further to create the tables and figures appearing in the paper.

# Benchmark Instances

This directory contains benchmark instances used in the computational experiments.

## xLOLIB

The subdirectories

```text
xLOLIB_150/
xLOLIB_250/
xLOLIB_300/
xLOLIB_500/
xLOLIB_750/
xLOLIB_1000/
```

contain instances derived from xLOLIB https://github.com/sgpceurj/EJOR2015.

The suffix indicates the number of vertices.

## MB

The directory

```text
MB/
```

contains Instances of Mitchell and Borchers.
